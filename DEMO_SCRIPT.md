# Report Designer Demo Script

**Total Runtime:** ~4-5 minutes
**Purpose:** Showcase AI-powered financial report generation for Canadian Big 6 banks

---

## Pre-Demo Checklist

- [ ] Backend running (`uvicorn src.api.main:app --port 8000`)
- [ ] Frontend running (`npm run dev` in frontend folder)
- [ ] Browser open to `http://localhost:5173`
- [ ] Screen recording software ready
- [ ] Clear any test data if needed

---

### Action
Start on the **Templates List** page (home page)

### Subtitle
```
Report Designer: AI-Powered Financial Report Generation
```

### What to Show
- The clean template list interface
- Existing templates visible

## SCENE 2: Create New Template (0:15 - 0:35)

### Action
1. Click the **"+ New Template"** button (top right)
2. Fill in the modal:
   - **Name:** `Q4 2024 Canadian Banks Analysis`
   - **Description:** `Quarterly performance review of Big 6 banks`
   - **Format:** PDF
   - **Orientation:** Landscape
3. Click **"Create Template"**

### Subtitle
```
Creating a new report template in seconds
```

### What to Show
- The modal appearing
- Quick form fill
- Template creation

---

## SCENE 3: Template Workspace Overview (0:35 - 0:55)

### Action
1. You're now in the workspace
2. Slowly pan across the interface:
   - **Left sidebar:** Section navigation
   - **Center:** Section editor with subsection tiles
   - **Right panel:** Subsection configuration
   - **Bottom:** AI Chat panel

### Subtitle
```
The workspace: Sections, Subsections, and AI Assistant
```

### What to Show
- The three-panel layout
- Empty state prompting to add sections

---

## SCENE 4: Add First Section (0:55 - 1:15)

### Action
1. Click **"+ Add Section"** in the left sidebar
2. A new section appears as "Section 1"
3. Click on the section title and rename it to: `Executive Summary`
4. Notice: A default subsection "A" is automatically created

### Subtitle
```
Sections organize your report - each can span multiple pages
```

### What to Show
- Section appearing in sidebar
- Renaming the section
- Default subsection tile

---

## SCENE 5: Configure Subsection Instructions (1:15 - 1:50)

### Action
1. Click on the **Subsection A** tile to select it
2. In the **right configuration panel**, find the **Instructions** field
3. Type this instruction:
   ```
   Summarize the key financial highlights from Q4 2024 for Royal Bank (RY)
   and TD Bank (TD). Include revenue growth, EPS, and ROE. Keep it to
   3-4 bullet points with a professional executive tone.
   ```
4. Optionally add a **Note**:
   ```
   Boss prefers concise bullet points over paragraphs
   ```

### Subtitle
```
Instructions tell the AI exactly what content to generate
```

### What to Show
- The configuration panel
- Typing instructions
- The difference between Instructions (formal) and Notes (informal)

---

## SCENE 6: AI Content Generation (1:50 - 2:30)

### Action
1. Click the **"Generate Section"** button (green button in section header)
2. Watch the loading indicator on the subsection tile
3. Wait for content to appear (~5-10 seconds)
4. Review the generated content in the tile

### Subtitle
```
AI generates content using real earnings data and transcripts
```

### What to Show
- The generate button
- Loading state animation
- Content appearing in the tile
- Quality of generated content (financial data, proper formatting)

---

## SCENE 7: Add More Subsections (2:30 - 2:55)

### Action
1. Click **"+ Add Subsection"** button (dashed border at bottom)
2. A new **Subsection B** appears
3. Click to select it
4. Add instructions:
   ```
   Create a comparison table of CET1 capital ratios for all Big 6 banks
   in Q4 2024. Highlight which banks exceed regulatory requirements.
   ```

### Subtitle
```
Subsections (A, B, C) stack vertically - add as many as needed
```

### What to Show
- Adding subsection B
- The A, B labeling system
- Different instruction for different content type

3. Scroll to see how sections and subsections render
4. Click back to **"Edit"** mode

Preview shows how your report will look when exported

## Key Talking Points (for live demo or voiceover)

If adding voiceover, emphasize these points:

1. **"Real financial data"** - The AI has access to actual earnings transcripts and financial metrics from Canadian Big 6 banks

2. **"Structured workflow"** - Sections and subsections provide organization while allowing flexibility

3. **"AI + Human collaboration"** - Instructions guide AI, Notes preserve context, humans review and refine

4. **"Professional output"** - PDF export produces presentation-ready documents

5. **"Iterative refinement"** - Use chat to make changes, regenerate sections, build incrementally
